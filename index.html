O que é o RJSEletrico?

O RJSEletrico é um software criado pelo Eng. Ricardo José da Silva para facilitar a elaboração de projetos elétricos de loteamentos. Adaptável às exigências específicas de cada concessionária, ele automatiza cálculos, geração de listas, croquis e integração com CAD.

Mais de 60 cópias já vendidas no Mercado Livre!

Funcionalidades Principais

  • Adaptação automática para normas de cada concessionária
  • Cálculo de carga, demanda, quedas e transformadores
  • Lista automática de materiais por poste e circuito
  • Geração de croquis e tabelas técnicas
  • Integração com GStarCAD, ZWCAD via API
  • Gera Diagrama Unifilar
  • Gera FQT folha de calculo de queda de tensao
  • Desenha Blocos de postes cabos, Basta indicar caminho da rede
  • Posicionamento automatico postes segundo norma
